# Tip Calculator

## Description

This project is a Python tip calculator. It is a simple program that helps the user calculate a tip and the final total of a bill. The project is organized in different modules to separate the interface and the calculations.

## How it works

1. The program asks the user to enter the price of a product or service.
2. The system calculates the tip using the function `calcular_propina`.
3. The program calculates the final total using the function `calcular_total`.
4. The result shows the tip and the total amount to pay.
